# 变量

```rust
// 定义时自动指定类型
let a = 10;
let a = 5; // 合法, 甚至可以改类型
```

决定类型后, 下面操作禁止

~~a = "abc";~~ 原因: a的类型已确定
~~a = 4.56;~~      : a是i32, 4.56自动强制转换i32会丢失精度, Rust语言不允许精度有损失的自动数据类型转换
~~a = 456;~~       : a不是mutable的

```rust
// 指定不可变变量的类型
let a: i32 = 20;
let b = 30_i32;
// 可变变量
let mut c = 30i32; // mutable: 可变的
c = 20;
let d = (add(a, b);
```

# 常量

```rust
const a: i32 = 123;
```

a 完全不可变
~~const a: i32 = 123;~~ 这种写法会报错重定义
