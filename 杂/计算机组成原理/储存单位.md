## Embedded system

An embeded system is a **computer system**

## proper noun of a conceptual nature

- Personal Mobile Device,PMD:个人移动设备
- cloud computing:云计算
- Warehouse Scale Computer,WSC:仓储规模计算机
- Software as Service,SaaS:软件即服务

    - Delivering software and data as a service over a network.
    - Instead of running all the binary code on the local device, it is usually run by logging on to a remote server through a small program such as a browser running on the local client.
    - The typical example is Web searching and social networking.

## What determines **program performance**

- 算法:决定了源码级语句的数量和I/O
- 编程语言、编译器和体系结构:操作的决定了每条源码级语句对应的计算机指令数拭数量
