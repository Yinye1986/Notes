# 简单开放某个端口(不安全)

sudo firewall-cmd --permanent --add-port=4242/udp
sudo firewall-cmd --permanent --remove-port=4242/udp

# 创建一条富规则

```shell
sudo firewall-cmd --permanent --add-rich-rule='rule \
family="ipv4" \
source address="192.168.1.100" \
port protocol="udp" \
port="4242"
accept'
```

分别是
1. 协议版本限制(ipv4)

2. ip 地址限制; 这里使用 source address="192.168.1.0/24" 的方式可限制一个网段; source mac="00:11:22:33:44:55" 可限制一个物理地址
3. 端口协议限制
4. 端口限制
5. 处理方式 accept|drop|reject
