# 定义
$ 若对任意 x_1 x_2 \in I, \thinspace 当 x_1  x_2 ​ 时, \thinspace 总有 f(x_1) \le f(x_2), \thinspace 使f(x_1​) \le f(x_2​)，则称 f 在 I 上单调递增 $
# 证明单调性

## 定义法

适用于任何函数

证明步骤:

1. $ 任取 x_1,x_2 \in I, \thinspace x_1,x_2​\in I, \thinspace 且设 x_1 < x_2 $
2. $ 观察差值 f(x_2)−f(x_1) 的符号 $
3. $ 若能证明 f(x_2)−f(x_1) \ge 0 或 > 0, \thinspace 则单调递增 $
4. $ 若 \le 0, \thinspace 或 < 0，则单调递减 $


## 导数法

适用于可导函数

$ 若 f'(x) \ge 0 对所有 x \in I 成立, \thinspace 则 f 在 I 上单调递增 $

$ 若 f'(x) > 0 对所有  x \in I 成立, \thinspace 则 f 在 I 上严格单调递增 $

$ 类似地，f'(x) \le 0 或 < 0, \thinspace 则单调递减或严格递减 $

### 注意

$ f'(x) \ge 0 且导数在任意子区间不恒为零也算是严格递增 $

$ 仅在个别点 f'(x) = 0 , 如 f(x) = x^3 在 x=0 处, \thinspace 不影响严格单调性 $

## 单调性传递法则

### 复合函数中的传递

$ 增 \circ 增 = 增 $

$ 增 \circ 减 = 减 $

$ 减 \circ 减 = 增 $

### 线性组合中的传递

$ 若 f,g 递增, \thinspace 且 a,b \ge 0, \thinspace 则 af+bg 递增 $

### 反函数中的传递

$ 若 f 严格单调且连续, 其 f^{-1} 也严格单调且同向 $
