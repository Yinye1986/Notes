# 查看当前数据库的表

```sql
SHOW TABLES;
```

# 创建表

```sql
CREATE TABLE tb_name (
   id INT PRIMARY KEY AUTO_INCREMENT,
   name VARCHAR(128),
   age INT
) CHARACTER SET utf8;
```

# 修改表属性

## 增加字段

```sql
ALTER TABLE tb_name ADD COLUMN new_column INT;
```

## 修改字段数据类型

```sql
ALTER TABLE tb_name MODIFY COLUMN new_column VARCHAR(128);
```
