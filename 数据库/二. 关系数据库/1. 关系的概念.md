# 基本概念
域: 域是一组同类型的值的集合

D1 = 课程{语文, 数学, 英语}, D2 = 成绩{优, 良, 及格}

笛卡尔积: 域的结合

D1xD2 = {(语文, 优), (语文, 良), (语文, 及格), (数学, 优), ..., (英语, 及格)}

关系: 笛卡尔积 D1xD2xD3x...xDn 的子集叫做在域 D1,D2,D3,...,Dn 上的元关系, 表示为
R(D1, D2, D3, ..., Dn), R表示关系名字, n是关系的 **目** 或者 **度**

| 科目 | 成绩等级 |
|------|----------|
| 语文 | 优       |
| 语文 | 良       |
| 语文 | 及格     |
| 数学 | 优       |
| 数学 | 良       |
| 数学 | 及格     |
| 英语 | 优       |
| 英语 | 良       |
| 英语 | 及格     |

上面的二维表中, 一行为一个 **元组**, 元组中的一个取值叫做一个 **分量**, 一列为一个 **属性**

而一个关系对应一张 **二维表**

# 键

能唯一标识一个元组, 称该属性为 **候选键**, 该属性为 **主属性**, 不包含主属性的属性为 **非主属性**

从候选键中任选一个作为 **主键**
